# HUG Terraform Challenge

This repository is my personal journey through the HashiCorp User Group Lagos/Ibadan Terraform Challenge.

Over the next 4 weeks, I will be learning by doing, building hands-on Terraform projects, attending live sessions every Saturday, and growing together with other participants as we strengthen our Infrastructure as Code skills.

## What I Hope to Learn

- How to write and use Terraform configuration confidently
- How to provision infrastructure on Azure
- How to apply Infrastructure as Code concepts in real projects
- How to solve practical cloud deployment challenges step by step

## Repository Contents

This repository contains the Terraform projects I am working on during the challenge, including:

- A basic Azure-based web server deployment
- Terraform configuration files, variables, outputs, and setup scripts

## How I Will Use This Repository

I will use this space to document my progress, keep my project files organized, and share the results of what I learn throughout the challenge.

## Getting Started

1. Open the project folder you want to explore.
2. Review the Terraform files.
3. Run `terraform init` to initialize the provider.
4. Run `terraform plan` and `terraform apply` to deploy the infrastructure.
